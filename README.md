# BipartisanCurious Prototype

A prototype dashboard comparing current U.S. House members by how often they vote with the opposite party's majority on party-split roll calls.

## Current prototype

The dashboard is built as a static GitHub Pages site from the `docs/` folder.

Current outputs include:

- 119th Congress voting alignment map
- current-member district map
- top bipartisan members table
- methodology/stat cards

## Data notes

The current prototype uses Clerk of the House XML roll call files for member votes. The XML includes Bioguide IDs, which are used to join member votes to profile and district data.

Raw scrape outputs are not committed to the repository.
